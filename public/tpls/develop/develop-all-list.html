HTTP协议

HTTP（HyperText Transport Protocol）是超文本传输协议的缩写，是互联网上应用最为广泛的一种网络协议。它用于传送WWW方式的数据，协议采用了请求/响应模型。目前接口短信主要使用此协议，详见接口文档。

CMPP协议

CMPP协议（China Mobile Peer to Peer） 是中国移动通信互联网短信网关接口协议。CMPP协议以TCP/IP作为底层通信承载。在有较高的安全性需要时，可选用TLS层进行安全传输，目前主要有2.0和3.0两个版本。
下载2.0版本 下载3.0版本

SGIP协议

SGIP是Short Message Gateway Interface Protocol的英文缩写，是中国联通短消息网关系统接口协议。SGIP协议是目前最流行的短信接口之一，相对于CMPP还是比较宽松的，目前常用的是SGIP1.2版本。
下载1.2版本

SMGP协议

SMGP协议（Short Message Gateway Protocol）由中国电信研发，本标准介绍综合信息系统短消息网关和内容提供商（CP）之间、短消息网关和短消息网关之间的通信协议，也是目前常见的接口协议之一。
